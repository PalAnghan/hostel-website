# UrbanNest - Hostel Website

A modern full-stack hostel booking website built using:

##  Tech Stack
- React + Vite
- Tailwind CSS
- Node.js
- Express.js
- MongoDB

##  Features
- Responsive modern UI
- Contact form with MongoDB storage
- Admin dashboard to manage messages
- Booking page
- Protected admin login

##  Installation

### Frontend
cd urban-nest
npm install
npm run dev

### Backend
cd backend
npm install
node server.js
